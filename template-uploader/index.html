Template Uploader

Etsy Listing Management Tool

Overview

Template Uploader is a personal-use application built to manage digital product listings in the Etsy shop "TheRogueSmith" (etsy.com/shop/TheRogueSmith). It automates the process of creating, updating, and organizing listings for Notion templates and Google Sheets dashboard products.

This application is used exclusively by the shop owner to manage their own listings. It does not access any third-party shops or user data.

Features

  • Batch Listing Creation — Creates draft listings with titles, descriptions, tags, pricing, and digital file attachments for template products
  • Shop Section Management — Organizes listings into categorized shop sections (e.g., "Notion Budget & Finance Templates", "Google Sheets Finance Dashboards")
  • Listing Image Upload — Attaches product mockup images to listings
  • Shop Profile Updates — Updates shop title, announcement, and digital sale messages to keep the storefront current
  • Inventory Management — Updates listing quantities and pricing in bulk

API Endpoints Used

  • PUT /v3/application/shops/{shop_id} — Update shop details
  • POST /v3/application/shops/{shop_id}/sections — Create shop sections
  • POST /v3/application/shops/{shop_id}/listings — Create draft listings
  • PUT /v3/application/shops/{shop_id}/listings/{listing_id} — Update listings
  • POST /v3/application/shops/{shop_id}/listings/{listing_id}/images — Upload images
  • POST /v3/application/shops/{shop_id}/listings/{listing_id}/files — Upload digital files
  • GET /v3/application/shops/{shop_id}/listings — Retrieve listings

Access Level

Personal use only. This application accesses a single shop owned and operated by the developer. No third-party shop data is accessed or stored.

OAuth Scopes

  • shops_r, shops_w — Read and update shop details and sections
  • listings_r, listings_w — Read, create, and update listings
  • listings_d — Delete draft listings if needed

Data Handling

  • No user data is collected, stored, or shared
  • OAuth tokens are stored locally on the developer's machine only
  • The application runs locally via command line — it is not a hosted web service

Compliance

This application complies with Etsy's API Terms of Use:

  • Section 3 — Application only accesses the developer's own shop data
  • Section 5 — No scraping, no circumventing API, no unauthorized data collection
